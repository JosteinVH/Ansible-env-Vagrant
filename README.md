# Ansible-env-Vagrant


## Requirements
* [Virtual Box] (https://www.virtualbox.org/wiki/Download_Old_Builds_6_0) Version 6.0.14 
* [Vagrant] (https://releases.hashicorp.com/vagrant/2.2.6/vagrant_2.2.6_x86_64.msi) 2.2.6
* Ansible Version 2.9.3

## Usage
